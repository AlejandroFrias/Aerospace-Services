    authenticate(username, password) click to toggle source

    Description¶ ↑

    Authenticates a given username/password combo. If it is a new username a new user is created. Returns the user if authentication passes, otherwise returns false.

    Parameters:¶ ↑

    username - the name of the user (string) password - the given password (string)

    # File app/models/user.rb, line 33
def self.authenticate(username, password)
  if username.blank?
    return false
  end

  user = User.create_with(password: password).find_or_create_by(name: username)
  if user.password == password
    return user
  else
    return false
  end
end
